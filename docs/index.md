# Web3aggregator

**Aggregated Web3 infrastructure for builders.**

Web3aggregator unifies access to RPC nodes, indexing, real-time event
streams, and aggregated on-chain data behind a single, reliable API
surface. Connect once — read from many.

## What you can do

- Query EVM RPC endpoints with automatic failover across providers
- Subscribe to real-time event streams over WebSocket
- Read aggregated on-chain state without managing your own infrastructure
- Build integrations against a stable, versioned API
